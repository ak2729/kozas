원본: PowerPoint 슬라이드 (Mar. 2025, Giyoung Kim)

프로젝트 개요

IEC61850 기반 계측에서 측정 상한(Voltage/Current)과 Deadband 설정으로 인해 고전압/고전류 구간이 보고되지 않는 문제를 CID 파일 파라미터 튜닝으로 해결한 사례입니다.

문제 (Trouble)

  • 6.6kV SWGR : It is not possible to monitor voltages above 1000v.
  • 480V SWGR : It is not possible to monitor ampere above 1000A![][image_bOjMdcGYcY707cUfj7rAFg==]

(슬라이드 2 요약)

원인/배경

  • Default CID settings: deadband=1000, range=1000 (need to monitor >6600V and ±10V fluctuations)
  • Deadband formula: db * (rangeC.max – rangeC.min)/100000. Example yields 10V threshold

Deadband = db × (rangeC.max – rangeC.min) / 100000

(슬라이드 3–4 요약)

  • Open CID in Notepad++; locate FFT voltage/current/power entries (Ctrl+F)
  • Modify: db 1000 → 100, Max 1000 → 10000; save and verify in CET850
